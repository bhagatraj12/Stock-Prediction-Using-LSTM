# Stock-Price-Prediction-Using-LSTM-ANN-
This project presents an advanced **Stock Price Prediction System** powered by **Long Short-Term Memory (LSTM)** networks. By leveraging historical stock data and deep learning techniques, the model captures complex market patterns to forecast future price movements with improved accuracy.

Designed with a focus on both performance and usability, the system integrates data preprocessing, model training, and interactive visualization into a seamless workflow. It enables users to analyze trends, compare predicted vs actual values, and gain meaningful insights into stock behavior.

This project demonstrates the practical application of **Deep Learning in Financial Forecasting**, combining data science, machine learning, and intuitive UI design.
